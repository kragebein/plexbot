Tilgjengelige kommandoa:
* tag - Game of Tag
* rejoinchat - Om plexbota dætt ut.
* .search - Søk etter film/serie. .search game of thrones
* .request - Request en film/serie: .request tt12345
Plugins du kan bruk:
* youtubeadd - legg tell film/serie ved å link yt.
* imdbadd - legg tell film/serie ved å link imdb.
